## 簡介

從零開始自學 C++ 中，這裡存放一些我學習過程中寫出來的東西，也希望能幫助到其他正在學習 C++ 的人。

如果需要找特定題目的解答，請善用搜尋功能

![alt text](https://github.com/poflygogo/randomBeginnerPythonCode/blob/main01/05%20misc/image%20for%20readme%2001.png)

如果有任何疑問都可以利用 discussion 說。

有需要 python 版的 zerojudge 答案的話可以前往另一個倉庫: [randomBeginnerPythonCode](https://github.com/poflygogo/randomBeginnerPythonCode)

## 使用的工具

- Visual Studio Code(VS Code): 主要使用這個編輯/編譯 c++ 程式碼
- notepad++

- 編譯器: `MinGW-W64 G++ 15.2.0`, 以 `C17` 標準為主

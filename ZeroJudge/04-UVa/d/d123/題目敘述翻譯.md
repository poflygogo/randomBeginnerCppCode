# UVa 11063 - B2-Sequence

## 問題描述

給定一個序列 $S = \{s_1, s_2, s_3, ...\}$，其中 $s_1 < s_2 < s_3 < ...$，若該序列中任意兩個元素（包括自己）的和 $s_i + s_j$（其中 $i \leq j$）都不重複，則稱此序列為 *B2-Sequence*。

換句話說，對於所有 $i \leq j$ 和 $k \leq l$，如果 $s_i + s_j = s_k + s_l$，那麼必然 $i = k$ 且 $j = l$。

例如，序列 $\{1, 2, 3, 5\}$ 不是 B2-Sequence，因為 $1 + 5 = 2 + 4$，但序列中並沒有 4，所以這個例子不成立。更正確的例子是 $\{1, 2, 4, 8\}$，其所有可能的 $s_i + s_j$ (i ≤ j) 組合為 $\{1+1=2, 1+2=3, 2+2=4, 1+4=5, 2+4=6, 4+4=8, 1+8=9, 2+8=10, 4+8=12, 8+8=16\}$，這些和都沒有重複，所以 $\{1, 2, 4, 8\}$ 是一個 B2-Sequence。

你的任務是判斷給定的序列是否為 B2-Sequence。

## 輸入

輸入包含多組測試資料。每組測試資料的第一列包含一個整數 $n$ ($1 \leq n \leq 100$)，代表序列的長度。
第二列包含 $n$ 個整數 $s_1, s_2, ..., s_n$。

所有輸入的數字 $s_i$ 都滿足 $1 \leq s_i \leq 20000$。

## 輸出

對於每組測試資料，輸出一列 "Case #X: Y"，其中 $X$ 是測試資料的編號（從 1 開始），$Y$ 是 "It is a B2-Sequence." 或 "It is not a B2-Sequence."。

在每組測試資料的輸出後，請輸出一個空行。

## 範例輸入

```
5
1 2 3 5 7
5
1 2 3 4 5
```

## 範例輸出

```
Case #1: It is a B2-Sequence.

Case #2: It is not a B2-Sequence.
```

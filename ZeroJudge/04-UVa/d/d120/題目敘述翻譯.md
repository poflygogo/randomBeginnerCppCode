# UVa 10699 - Count the factors

## 問題描述

給定一個正整數 $N$，你的任務是計算 $N$ 有多少個 *不同的* 質因數。

例如：
- 數字 23 是一個質數，它只有 1 個質因數（就是它自己）。
- 數字 9 是 $3 \times 3$，它只有 1 個 *不同的* 質因數（3）。
- 數字 30 是 $2 \times 3 \times 5$，它有 3 個 *不同的* 質因數（2, 3, 5）。

## 輸入

輸入包含多組測試資料，每組測試資料一列。
每一列包含一個整數 $N$，其中 $2 \leq N \leq 1000000$。
輸入以一列包含 0 結束。

## 輸出

對於每組測試資料，輸出一列，格式為 "$N : F$"，其中 $N$ 是輸入的數字，$F$ 是 $N$ 的不同質因數的個數。

> 譯者註：注意冒號和空格。

## 範例輸入

```
23
9
30
0
```

## 範例輸出

```
23 : 1
9 : 1
30 : 3
```

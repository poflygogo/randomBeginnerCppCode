# UVa 583 - Prime Factors

## 問題描述

任何大於 1 的正整數都可以被表示為質數的乘積。例如：
$$
225 = 3 \times 3 \times 5 \times 5
$$
$$
1001 = 7 \times 11 \times 13
$$
$$
10000 = 2 \times 2 \times 2 \times 2 \times 5 \times 5 \times 5 \times 5
$$

你的任務是計算一個整數的質因數分解。

## 輸入

輸入包含多組測試資料，每組測試資料一列。
每一列包含一個整數 $n$，其中 $-2^{31} < n < 2^{31}$。
輸入以一列包含 0 結束。

## 輸出

對於每個非零的輸入整數 $n$，輸出一列，格式為 "$n = f_1 \times f_2 \times ... \times f_k$"，其中 $f_1 \leq f_2 \leq ... \leq f_k$ 是 $n$ 的質因數。如果 $n$ 是負數，則第一個因數 $f_1$ 應該是 -1。

請注意：
- 數字 1 或 -1 沒有質因數，不需要輸出。
- 輸出格式中，乘號（$\times$）前後都有一個空格。

## 範例輸入

```
-190
-191
-192
-193
-194
195
196
197
198
199
200
0
```

## 範例輸出

```
-190 = -1 x 2 x 5 x 19
-191 = -1 x 191
-192 = -1 x 2 x 2 x 2 x 2 x 2 x 2 x 3
-193 = -1 x 193
-194 = -1 x 2 x 97
195 = 3 x 5 x 13
196 = 2 x 2 x 7 x 7
197 = 197
198 = 2 x 3 x 3 x 11
199 = 199
200 = 2 x 2 x 2 x 5 x 5
```

# c705. Int32 to IPv4

分類: 原創/不分類題庫
出處: it's david codewars

## 內容

IPv4 使用 &32& 位元（4位元組）位址，因此位址空間中只有 $4,294,967,296$（$2^{32}$）個位址。

IPv4 位址可被寫作任何表示一個 $32$ 位元整數值的形式([維基百科](https://zh.wikipedia.org/wiki/IPv4))

現在請你將每一個整數，轉換為 IPv4 的格式。 

## 範例輸入

```
1875123557
1643788805
32
```

## 範例輸出

```
111.186.226.226
123.125.111.186
0.0.0.32
```

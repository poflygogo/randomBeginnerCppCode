# 迷路小鴨

## 題目內容

有很多小鴨正慢慢的游到高雄，在光榮碼頭附近上班的阿霞很開心的迎接他們的來臨。經過阿霞的調查，得知這些小鴨的體重一定會形成等差數列，但不一定會依照體重順序抵達。

可是很不幸的，有一些小鴨走失了，為了透過體重的估計找回這些可愛的小鴨，阿霞希望可以計算出包含走失的小鴨們，這些小鴨的體重最大的可能差距是多少，也就是小鴨的體重所形成的等差數列中，相鄰兩項之間的差(即公差)的最大可能值。

## 輸入說明

每個輸入檔第一列為一個正整數 $T$，表示共有 $T$ 筆測試資料。

每筆測試資料的第一列為一個正整數 $N$，表示共有 $N$ 隻可愛的小鴨平安到達。

第二列為 $N$ 個正整數$(C_{1}, C_{2}, C_{3}, ... ,C_{N})$，分別為依序抵達的這 $N$ 隻小鴨的體重，其中不會包含相同的數。

$T \le 100$； $1 \le N \le 10000$；$C_{1}, C_{2}, C_{3}, ... ,C_{N} .\le 2^{60}$

## 輸出說明

對每筆測試資料，請輸出一列，表示所有小鴨的體重所形成的等差數列中，公差的最大可能值。
